## Inventory Managment System

A multile file console based inventory managment and purchase order system

## Features
* Create, edit, and save products and vendors
* Create and recive purchase orders
* Search for products, vendors, or orders within the system
* Find low or out of stock items
* Export product and vendor data to excel spreadsheet

## Required Files
* main.py
* inventory_manager.py
* file_manager.py
* reports.py
* models.py

## Instructions
Input numbers in console related to  process you wish to execute and follow the instructions when adding data

## How Data is stored
Data releating to the products, vendors, or orders are all stored in thier own .json file

Anytime data is changed it saves it to the .json file ensuring the accessable data is always up to date

If no .json files exist when program starts new ones with default data are created

## Exporting to Excel
Information relating to vendors and products can be exported to a excel spreadsheet

Spreadsheets contain all information about products and vendors for easy review of all data
